# 📚 StudyPilot AI – Personalized Study Planner

## 📖 Overview

StudyPilot AI is an AI-powered study planning application built using **n8n**, **Google Gemini AI**, **Google Forms**, **Google Sheets**, and **Gmail**.

The application automatically generates a personalized study plan based on a student's subject, daily study hours, and exam date, then emails the plan to the student.

---

## ✨ Features

- AI-generated personalized study plans
- Google Forms for collecting student information
- Automatic workflow using n8n
- Google Gemini AI integration
- Stores study plans in Google Sheets
- Sends professional HTML emails automatically

---

## 🛠️ Technologies Used

- n8n
- Google Gemini AI
- Google Forms
- Google Sheets
